## Configuration Example

Below is a typical `preferences.json` configuration that optimizes the editor for high-performance 4K workflow. This file lives in `%APPDATA%\Corel\VideoStudio\2026\Ultimate\`. Adjust the values based on your hardware.

```json
{
  "preview_quality": "Full",
  "gpu_acceleration": true,
  "max_timeline_resolution": 4320,
  "auto_save_interval_seconds": 120,
  "render_codec": "H.265",
  "audio_device": "ASIO",
  "theme": "dark",
  "language": "en-US",
  "motion_tracking_accuracy": "High",
  "proxy_scale": 0.5,
  "undo_history_limit": 50,
  "export_bitrate": 50000,
  "frame_interpolation": "Optical Flow",
  "cloud_sync_enabled": false,
  "custom_ffmpeg_path": "C:\\Program Files\\FFmpeg\\bin\\ffmpeg.exe"
}
```

**Explanation of key parameters:**
- `gpu_acceleration`: Leverages NVIDIA CUDA or AMD VCE for hardware encoding.
- `frame_interpolation`: Uses AI-based optical flow for smoother slow-motion.
- `proxy_scale`: 0.5 means 50% proxy resolution for timeline editing; full quality on export.
- `custom_ffmpeg_path`: Allows integration with external FFmpeg for advanced workflows.

To apply, save the file and restart the editor. No compilation or terminal commands are required.

## Console Invocation

For advanced users, the VideoStudio engine exposes a command-line interface for batch processing and automation. Use the following invocation to render a project without launching the GUI:

```bash
VideoStudioCLI.exe --project "C:\Projects\MyEdit.vsp" --output "C:\Exports\Final_4K.mp4" --preset "YouTube 4K HDR" --overwrite
```

**Arguments:**
- `--project`: Path to the `.vsp` project file.
- `--output`: Destination file path.
- `--preset`: Render preset (e.g., `YouTube 4K HDR`, `Facebook 1080p`, `Custom H.265`).
- `--overwrite`: Force overwrite if output exists.
- `--verbose`: Enable detailed logging.
- `--threads`: Override CPU thread count (default: all cores).

You can also run a headless batch render of a folder:

```bash
for %%f in (C:\Projects\*.vsp) do VideoStudioCLI.exe --project "%%f" --output "C:\Exports\%%~nf.mp4" --preset "Quick 1080p"
```

*Note: The CLI tool is included inside the installation directory under `Tools\CLI\VideoStudioCLI.exe`. No additional setup is required.*
